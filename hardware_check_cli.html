How to collect hardware information in Linux??? 1. dmidecode - DMI table decoder 2. lsscsi - list SCSI devices (or hosts) and their attributes 3. lspci - list all PCI devices 4. hdparm provides a command line interface to various hard disk ioctls supported by the stock Linux ATA/IDE device driver subsystem. 5. lshw is a small tool to extract detailed information on the hardware configuration of the machine. 6. lscpu - display information on CPU architecture 7. lsusb - list USB devices 8. lsdev - display information about installed hardware 9. lsblk lists information about all or the specified block devices. The lsblk command reads the sysfs filesystem to gather information. 10. [Ubuntu]inxi is a command line system information script built for for console and IRC. It is also used for forum technical support, as a debugging tool, to quickly ascertain user system configuration and hardware. inxi shows system hardware, CPU, drivers, Xorg, Desktop, Kernel, GCC version(s), Processes, RAM usage, and a wide variety of other useful information. 11. hpacucli - HP Command Line Array Configuration Utility, Google to find more about the usage of it. 12.
